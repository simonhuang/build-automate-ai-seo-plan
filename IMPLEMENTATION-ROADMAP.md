# Implementation Roadmap

## KPI Targets

| Metric | Launch | 3 months | 6 months | 12 months |
|---|---|---|---|---|
| Organic sessions/mo | 0 | 500–1,500 | 3,000–8,000 | 15,000–40,000 |
| Keywords ranking (any position) | 0 | 50–100 | 300–600 | 1,000+ |
| AI Overview appearances | 0 | 2–5 | 10–20 | 30+ |
| Backlinks (referring domains) | 0 | 10–25 | 50–100 | 150–300 |
| Blog posts published | 0 | 16–24 | 40–56 | 80–100+ |

---

## Phase 1 — Foundation (Weeks 1–4)

Goal: get the right pages indexed with the right signals from day one.

### Pages
- [ ] Homepage copy anchored to "build + automate with AI"
- [ ] `/build` pillar page live
- [ ] `/automate` pillar page live
- [ ] `/method` page live (4-step system with `HowTo` schema)
- [ ] `/work` with minimum 3 case studies (1 build, 1 automation, 1 combined)
- [ ] `/learn` + `/sprint` conversion pages live
- [ ] `/about` with full E-E-A-T author signals

### Technical
- [ ] HTTPS confirmed
- [ ] Google Search Console verified + sitemap submitted
- [ ] Google Analytics / Plausible configured
- [ ] `robots.txt` configured (index all, block `/admin`)
- [ ] `llms.txt` at site root for AI crawlers
- [ ] OpenGraph tags on all pages
- [ ] Core Web Vitals baseline measured (target LCP < 2.5s)

### Schema
- [ ] `Organization` + `WebSite` on homepage
- [ ] `HowTo` on `/method`
- [ ] `Course` on `/learn`
- [ ] `Person` on `/about`

---

## Phase 2 — Content Engine (Weeks 5–12)

Goal: establish topical authority in both keyword clusters.

### Content
- [ ] 2 blog posts/week (alternate Build / Automate pillars)
- [ ] 1 case study published per month
- [ ] First GEO-targeted post live ("What's the difference between AI tools and AI systems?")
- [ ] `/tools` free resource page live

### SEO
- [ ] Internal linking audit: every post links to its pillar page
- [ ] `BlogPosting` schema on all published posts
- [ ] Email capture on high-traffic posts (funnel into course/sprint)
- [ ] First backlink outreach: submit to AI/no-code newsletters and roundups

### Tracking
- [ ] Rank tracking set up for Tier 1 keywords
- [ ] Search Console impressions reviewed weekly
- [ ] AI Overview monitoring started (manual or via tool)

---

## Phase 3 — Authority (Weeks 13–24)

Goal: build domain authority and expand keyword coverage.

- [ ] Guest posts / podcast appearances in adjacent niches (productivity, no-code, solopreneur)
- [ ] Original data post published: personal metrics ("across X builds, average time to ship was...")
- [ ] Expand `/work` to 10+ case studies
- [ ] Start targeting Tier 2 keywords with dedicated posts
- [ ] `/tools` page expanded into a comparison table (link magnet)
- [ ] Repurpose YouTube/social content into blog posts for topical velocity
- [ ] Review and update top 5 posts with fresher data + internal links

---

## Phase 4 — Dominate (Months 7–12)

Goal: own the "build + automate with AI" keyword space.

- [ ] Target "AI automation" head term with full content cluster (10+ supporting posts)
- [ ] Launch "State of AI Building" original research post (high citation potential)
- [ ] Programmatic SEO consideration: `/blog/automate/[workflow-type]` at scale if applicable
- [ ] PR outreach for media mentions (builds E-E-A-T and domain authority simultaneously)
- [ ] Monitor and respond to AI Overview appearances for core terms
- [ ] Advanced schema: `ItemList` on `/tools`, `Review` on case studies

---

## Strategic Priority Note

> The phrase **"build + automate with AI"** is genuinely unowned right now. The first site to rank for that compound intent — not just "build with AI" or "automate with AI" — will own a fast-growing keyword cluster as AI tool adoption accelerates.
>
> Publishing 2 posts/week from day one and getting `/build` and `/automate` indexed as proper pillar pages is the highest-leverage move in the first 90 days.
